About the CLV Prediction Project

What is Customer Lifetime Value (CLV)?

Customer Lifetime Value (CLV) is a key metric used in business analytics to estimate the total revenue a business can expect from a customer throughout their relationship. It helps companies make strategic decisions regarding marketing, customer acquisition, and retention.

How CLV is Calculated

CLV is determined using the following formula:

CLV = Average Transaction Value × Purchase Frequency × Retention Period

  • Recency: Measures how recently a customer made a purchase.
  • Frequency: Indicates how often a customer makes purchases.
  • Monetary Value (Average Transaction Value): Represents the average amount spent per transaction.
  • Retention Period: The expected duration a customer continues buying from the business.

Project Overview

This CLV prediction system is built using Python, Flask, HTML, CSS, and machine learning. It allows users to:

  • Upload customer transaction history.
  • Analyze past purchase behaviors.
  • Predict Customer Lifetime Value based on historical data.
  • Calculate CLV for both existing and new customers.
  • Display detailed insights through an interactive user interface.

How the Project Works

  1. Data Upload: Users upload a transaction dataset containing customer purchase records.
  2. Data Processing: The system cleans and preprocesses the data to extract relevant customer behavior metrics.
  3. CLV Calculation: The machine learning model predicts CLV based on the extracted features.
  4. Results Display: The predicted CLV values are displayed along with insights on customer behavior.
  5. New Customer Prediction: Users can input new customer details to estimate their potential CLV.
